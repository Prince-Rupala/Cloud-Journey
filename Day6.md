# Day 6

## Learned

- > :- Replaces file contents with output of command
- >> :- Appends to file contents with output of command
- | :- Takes output from command A and send it to command B
- grep :- Search tool
- head :- Used to view first 10 lines of a file (by default)
- tail :- Used to view last 10 lines of a file (by default)

## Key Difference

> replaces file contents.

>> appends to file contents.

## Most Useful Command

- grep & |
